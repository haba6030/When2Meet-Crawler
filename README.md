# When2Meet-Crawler
This extracts the result of a when2meet link

## Before Usage
- Change the url link of the given main.py file

## How to use
1. Install Chrome
2. Install Python 3.9
3. Make Folder
`mkdir git`
4. Change Directory
`cd git`
5. Use Git Clone
`git clone `
7. Download Library
`cd Instagram-Image-Crawling`
`pip install -r requirements.txt`
9. Use the program
`python main.py`
